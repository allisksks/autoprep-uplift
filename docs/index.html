Зачем uplift-моделирование?

Классические модели предсказывают кто купит. Uplift-модели предсказывают чьё поведение изменится благодаря промо.

Как это работает

Структура пайплайна

Восемь шагов — LLM-агенты на ключевых этапах.

01
Валидация схемы guard
Проверяет treatment_col, outcome_col, user_id. Блокирует при ошибке.
02
Валидация данных checks
Проверка рандомизации (t-test/chi-square), детекция leakage, баланс групп.
03
LLM Агент — EDA и препроцессинг LLM
Генерирует fit_preprocess() + apply_preprocess(). Самопроверка, итеративное исправление ошибок. Без leakage.
04
Обучение 6 мета-лёрнеров models
DR / T-LGB / T-Ridge / X-learner / R-learner / Hurdle — все с early stopping.
05
CV + антиовёрфиттинг eval
k-fold CV с bootstrap CI. Permutation test. Repeated CV. Train/val gap. Learning curves.
06
Перебор всех комбинаций на holdout search
108 комбинаций: одиночные, пары, тройки, полный ансамбль — все стратегии взвешивания.
07
LLM Агент — выбор ансамбля LLM
Агент видит все 108 результатов, выбирает оптимальную комбинацию с объяснением.
08
Предсказание и экспорт output
Финальные uplift-scores. Экспорт predictions.csv. Параметры препроцессинга в JSON.
Поддерживаемые методы

6 мета-лёрнеров

От простого Ridge baseline до теоретически оптимального R-learner.

DR-learner
Doubly Robust
Состоятелен если хотя бы одна из nuisance-функций верна. Cross-fitting.
ψ = (μ₁−μ₀) + W(Y−μ₁)/e − (1−W)(Y−μ₀)/(1−e)
Kennedy (2023)
T-learner LGB
Два LightGBM
Отдельные LightGBM для treatment и control. Быстрый сильный baseline.
τ(x) = μ₁(x) − μ₀(x)
Künzel et al. (2019)
T-learner Ridge
Два Ridge
Самый быстрый линейный baseline. Минимальный train/val gap.
τ(x) = Ridge₁(x) − Ridge₀(x)
Künzel et al. (2019)
X-learner
Cross Learner
Двухступенчатая импутация. Эффективен при дисбалансе групп.
τ(x) = p·τ_t(x) + (1−p)·τ_c(x)
Künzel et al. (2019)
R-learner ★
Robinson Decomposition
Квази-оракульный: сходимость зависит только от сложности CATE.
L = E[(Y−m−(W−e)·τ)² · (W−e)²]
Nie & Wager (2021)
Hurdle
Zero-Inflated модель
Для ~90% нулей. Отдельные модели конверсии и суммы.
τ = P_t·E_t[Y|Y>0] − P_c·E_c[Y|Y>0]
Devriendt et al. (2022)
Антиовёрфиттинг

5 проверок качества

До обучения, в процессе CV и после выбора модели.

⚖️
Проверка рандомизации
t-test и chi-square по каждому признаку. Флаги несбалансированных признаков.
🔍
Детекция leakage
Признаки коррелирующие с treatment — post-treatment leakage.
🎲
Permutation test
H₀: модель ≤ случайного ранжирования. 200 перестановок. p < 0.05 = значимо.
📈
Learning curves
Uplift@K при росте обучающей выборки. Показывает нужно ли больше данных.
🔁
Repeated CV
3×3-fold CV с разными seeds. Более надёжная оценка чем однократный CV.

Ключевой вывод

CV ranking ≠ ensemble contribution

X-learner — последний по CV (lower CI = -0.0115), но в предыдущих запусках без FE входил в winning ensemble.

Это ключевой методологический инсайт: модель которая плохо работает поодиночке может добавлять ценный сигнал в комбинации с другими. Именно поэтому перебор всех 108 комбинаций важнее жадного выбора топ-3 по CV.
